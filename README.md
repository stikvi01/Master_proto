# Master Thesis Prototype – Information Design for Cognitive Overload

## Overview

This project is a prototype developed as part of a Master’s thesis exploring how interaction design (IxD) strategies can improve comprehension under conditions of cognitive overload. The focus is on designing information flows that better support patients and informal caregivers during cancer treatment.

The project is conducted in collaboration with St. Olavs Hospital in Trondheim.

## Research Focus

The prototype investigates whether established design principles can make complex information easier to understand and remember when users are mentally overstimulated.

The key theories and frameworks applied include:

* Cognitive Load Theory
* Mayer’s Multimedia Principles
* Usability Heuristics

## Concept

The prototype simulates a health information app designed to communicate critical treatment-related information.

It includes two contrasting design approaches:

1. **Optimized Design** – Applies IxD strategies to reduce cognitive load and improve clarity.
2. **Non-Optimized Design** – Intentionally violates these principles to create a higher cognitive burden.

This contrast allows for direct comparison during user testing.

## Prototype Features

* Example mobile interface for health information delivery
* Structured information flow tailored for patients and informal caregivers
* Visual and textual elements designed according to (and against) key IxD principles
* Parallel versions of the same content (optimized vs. non-optimized)

## User Testing Method

The prototype was evaluated through a **dual-task experiment**, where participants were:

* Exposed to the prototype while simultaneously performing an additional task
* Placed under cognitive strain to simulate real-world mental overload

This approach ensures that:

* Differences in comprehension are more pronounced
* The effectiveness of design strategies can be more reliably measured

## Research Goal

The goal is to determine:

* Which design elements improve understanding under cognitive stress
* How information flow can better support patients and caregivers in high-pressure situations
* Whether IxD strategies meaningfully reduce cognitive overload in healthcare contexts
